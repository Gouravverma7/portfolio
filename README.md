# Gourav Verma - Portfolio Website

A modern, responsive portfolio website built with React and Vite, showcasing my skills as a Full Stack Developer. The site features smooth animations, interactive components, and a clean design aesthetic.

## 🚀 Features

- **Responsive Design**: Optimized for all device sizes from mobile to desktop
- **Smooth Animations**: Powered by Framer Motion for engaging user interactions
- **Modern UI**: Clean, professional design with custom color scheme
- **Interactive Navigation**: Smooth scrolling navigation with active section highlighting
- **Contact Form**: Functional contact form for easy communication
- **Project Showcase**: Dedicated section highlighting key projects
- **Skills Display**: Visual representation of technical skills and expertise

## 🛠️ Tech Stack

- **Frontend**: React 19.1.1
- **Build Tool**: Vite 7.1.0
- **Styling**: Tailwind CSS 3.4.17
- **Animations**: Framer Motion (motion 12.23.12)
- **Icons**: React Icons 5.5.0, Lucide React 0.540.0
- **Smooth Scrolling**: React Scroll 1.9.3
- **Linting**: ESLint 9.32.0

## 🚀 Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd gv-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## 📜 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
